## Overview

This plugin provides production-ready long-term memory for AI agents using PostgreSQL with pgvector extension.

### Key Features

- **Semantic Search** - Find relevant memories using vector similarity
- **Auto-Capture** - Automatically store important information
- **Auto-Recall** - Inject relevant context into conversations
- **Multi-Provider** - Supports OpenAI and E5 embeddings

## Architecture

The plugin consists of:
- PostgreSQL database with pgvector extension
- E5 embeddings server (local, free)
- OpenClaw plugin interface
