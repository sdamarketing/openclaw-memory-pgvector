## [1.0.0] - 2026-02-14

### Added
- Complete conversation tracking system
- Semantic search using pgvector
- Multi-provider embeddings (OpenAI, E5-local, Z.AI)
- Auto-capture and auto-recall functionality
- CLI tools for memory management
- GDPR-compliant memory forget tool
- PostgreSQL 16+ with pgvector support

### Features
- Store requests, responses, reasoning, and files
- Vector similarity search
- Automatic context injection for agents
- Session tracking across multiple users
